Install this setup

dothub does not touch your machine. Your agent installs the setup after you approve the effects below, and installing runs those effects, so reviewing them is your responsibility.

In a Claude Code session with the dothub MCP server added, say

First time? Mint an API key on your Account page, then add dothub
